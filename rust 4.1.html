4.1 Structurer ses données

Quand plusieurs valeurs vont ensemble (nom, âge, email), regroupe-les dans une struct. Déclare-la puis crée une instance. Exemple : struct Utilisateur { nom: String, age: u8 } let u = Utilisateur { nom: String::from("Alice"), age: 25 }; Accède aux champs avec u.nom. C'est comme une fiche qui décrit un objet. Tu peux rendre certaines valeurs modifiables si l'instance est déclarée mut. C'est plus propre que de jongler avec dix variables séparées.
