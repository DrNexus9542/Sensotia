1.3 Installation et premier pas

Installe Rust via rustup.rs, un petit programme qui gère les versions. Ouvre un terminal et tape cargo new mon_projet. Cargo est l'outil tout-en-un de Rust : il crée, compile et lance tes projets. Tape cd mon_projet puis cargo run. Tu verras "Hello, world!" s'afficher. Le fichier principal s'appelle src/main.rs. C'est ici que tu écriras ton code. Félicitations, tu as compilé ton premier programme Rust sans encombre.
