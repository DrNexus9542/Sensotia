5.1 Conditions et if

L'instruction if fonctionne sans parenthèses autour de la condition : if x > 5 { ... } else { ... }. La condition doit être un booléen. Rust ne convertit pas automatiquement un nombre en vrai/faux. Tu peux aussi utiliser else if pour chaîner les tests. Rust autorise même if comme expression pour renvoyer une valeur : let note = if reussi { "A" } else { "B" };. Les deux branches doivent alors renvoyer le même type. C'est concise et sécurisée.
