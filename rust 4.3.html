4.3 Les énumérations

enum liste des variantes possibles pour une donnée. Exemple : enum Message { Quitter, Deplacer { x: i32, y: i32 }, Ecrire(String) }. Une variable de type Message vaut forcément une seule de ces variantes à la fois. Contrairement à d'autres langages, les variantes peuvent contenir des données. C'est puissant pour modéliser des états : soit on quitte, soit on se déplace avec des coordonnées, soit on écrit un texte. Le tout dans un seul type cohérent et vérifiable.
