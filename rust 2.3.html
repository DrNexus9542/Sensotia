2.3 Les fonctions

Une fonction regroupe des instructions sous un nom. Tu la déclares avec fn. Par exemple : fn dire_bonjour() { println!("Salut !"); }. Pour l'appeler, écris simplement dire_bonjour();. Une fonction peut aussi renvoyer une valeur avec ->. Exemple : fn addition(a: i32, b: i32) -> i32 { a + b }. Notez l'absence de point-virgule sur la dernière ligne : cela signifie "retourne cette valeur". Les paramètres doivent toujours avoir un type explicite.
