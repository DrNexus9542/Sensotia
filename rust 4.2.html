4.2 Implémenter des méthodes

Tu peux attacher des fonctions à ta struct avec impl. La première paramètre s'appelle self. Exemple : impl Utilisateur { fn presenter(&self) { println!("Je suis {}", self.nom); } }. Appelle-la avec u.presenter(). Tu peux aussi faire des méthodes qui modifient avec &mut self, ou des fonctions associées (constructeurs) avec Self. Cela organise ton code : les données et les comportements liés restent au même endroit, comme dans d'autres langages orientés objet.
