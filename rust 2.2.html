2.2 Les types de base

Rust doit connaître le type de chaque donnée. Heureusement, il devine souvent tout seul. Les entiers s'écrivent i32 (positifs et négatifs) ou u32 (positifs seulement). Les décimaux utilisent f64. Le texte simple se note entre guillemets simples pour un caractère (char) et doubles pour du texte (String ou &str). Le booléen s'appelle bool et vaut true ou false. Quand Rust n'est pas sûr, il te demande de préciser explicitement.
