5.3 Pattern matching avec match

match compare une valeur contre plusieurs modèles (patterns). C'est un switch surpuissant. Exemple : match x { 1 => "un", 2 => "deux", _ => "autre" }. Le compilateur vérifie que tous les cas sont couverts. Tu peux déstructurer des structs, des enum, vérifier des plages, ou lier des variables. C'est la clé pour travailler avec Option et Result. Il est impossible d'oublier un cas, ce qui élimine une grande source d'erreurs silencieuses présentent dans d'autres langages.
